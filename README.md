Homepage and web app repository for Feng Ling 「凌峰」 (https://orcid.org/0000-0002-1766-073X)

## Local Jekyll preview

- Recommended Ruby: see `.ruby-version` (`3.2.0`).
- If you’re on macOS system Ruby (often `2.6.x`), install gems into the repo to avoid `sudo`:
  - `bundle install --path vendor/bundle`
  - `bundle exec jekyll serve --incremental --host 127.0.0.1 --port 4000`
